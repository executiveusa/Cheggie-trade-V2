# Browser Control Module

This module provides a reusable workspace for browser automation orchestration using:
- Browser Harness (preferred for open-ended resilient tasks)
- Chrome DevTools MCP (preferred for debugging/inspection/perf)
- Playwright (preferred for deterministic repeatable flows)
- Selenium (fallback for legacy WebDriver cases)

See `../BROWSER_CONTROL.md` for operator instructions and safety rules.
